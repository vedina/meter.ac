Query meter.ac with python code

Mar 28, 2020 • Nina Jeliazkova • 5 min read

meter.ac   weather

METER.AC https://meter.ac/ is an environmental monitoring network, developed in Bulgaria. The aim of this open monitoring network is to observe a broad range of natural processes on a long time scale with a high time resolution at evenly distributed ground stations. Some of the values we monitor are the atmospheric temperature, humidity and pressure, particulate matter, beta and gamma radiation, wind direction and speed, solar radiation, rainfall, lightning and thunderstorms, etc. We perform also indoor measurements by monitoring radon, CO2 and VOXs concentration.

Raw measurement data as well as some derivatives and statistics are freely accessible on our web site under the CC0 license.

import requests
import pandas as pd
import io
query= "select * from box where location = '{}' and time > now() - {}".format(location,time_interval)
payload = {'db': 'meter', 'u': client, 'p' : p, 'q': query}
headers = {'Accept': 'application/csv'}
r = requests.get(url, payload, headers=headers)


if r.status_code == 200:
    #print(r.content)
    df=pd.read_csv(io.StringIO(r.content.decode('utf-8')))
    display(df)
else:
    print(r.status_code)
